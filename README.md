A microservices-based inventory optimization system using Spring Boot, React.js, OpenAI, and AWS EKS.

## Services
- **inventory-service** - Manages products and stock levels (Port 8080)
- **ai-prediction-service** - OpenAI-powered stockout predictions (Port 8081)
- **api-gateway** - Routes and authenticates requests (Port 8082)
- **frontend** - React.js dashboard (Port 3000)

## Tech Stack
Java 17, Spring Boot, React.js, PostgreSQL, Docker, Kubernetes (EKS), Jenkins
